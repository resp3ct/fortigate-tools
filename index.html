Verification Commands

    

# ── BGP ──────────────────────────────────────────────
get router info bgp summary
get router info bgp neighbors <neighbor-ip> received-routes
get router info bgp neighbors <neighbor-ip> advertised-routes
get router info routing-table bgp

# ── IPsec ────────────────────────────────────────────
get vpn ipsec tunnel summary
diagnose vpn ike gateway list
diagnose vpn tunnel list

# ── Connectivity ────────────────────────────────────
execute ping-options source <local-tunnel-ip>
execute ping <remote-tunnel-ip>

# ── Routing ─────────────────────────────────────────
get router info routing-table all
get router info routing-table details <prefix>

# ── SD-WAN ──────────────────────────────────────────
diagnose sys sdwan health-check
diagnose sys sdwan member
diagnose sys sdwan service
get system sdwan

# ── Policies ────────────────────────────────────────
show firewall policy

# ── BGP Debug (disable after!) ──────────────────────
diagnose ip router bgp level info
diagnose ip router bgp all enable
diagnose debug enable
diagnose debug disable
diagnose ip router bgp all disable
